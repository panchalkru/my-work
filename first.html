What is WebDesign?

HTML

HTML stands for HyperText Markup Language.it is used on the frontend and gives the structure to the webpage which you can style using CSS and make interactive using javascript.

1. The DOCTYPE html tag declaration defines that this document is an HTML5 document.

2. The html tag element is the root element of an HTML page.

3. The head tagelement contains meta infromation about the HTML page.

4. The title tag element specifies a title for the HTML page(which is shown in the browser's title bar or in the page's tab).

5. The body tag element defines the document's body,and is a container for all the visible contents,such as headings,paragraphs,hyperlink,tables,lists,etc.

6. The h1 tagelement defines a large heading.

7. The p tagelement definte a paragraph.

CSS

CSS or Cascading Style Sheets is the language used to style the frontend of any website. CSS is a cornerstone technology of the World Wide Web, alongside HTML and javaScript.

1. CSS stands for Cascading Style Sheets.

2. CSS describes how HTML elements are to be displayed on screen,paper, or in other media.

3. CSS saves a lot of work. it can control the layout of multiple web pages all at once.

4. External stylesheets are stored in CSS files.

Bootstrap

Quickly design and customize responsive mobile-first sites with Bootstrap,the world's most popular front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components,and powerful JavaScript plugins.

1. Bootstrap is a free front-end framework for faster and easier web development.

2. Bootstrap includes HTML and CSS based design templets for typography,froms,buttons,tables.navigation,modals, image carousels and many other,as well as optional JavaScript plugins.

3. Bootstrap also gives you the ability to easily create responsive designs.
